Proiecte Data Science

Proiecte Data Science utilizate cu studenții în procesul educațional

Teoria probabilității și statisticii - un proiect educațional care are drept scop studierea noțiunilor de date, probabilități, distribuții și statistici necesarea în procesul de analiza, prelucrare și manipulare a datelor spre a fi utilizate în elaborarea modelelor Machine Learning (detalii...)
Baze de date MySQL - un proiect educațional care are drept scop studierea noțiunilor de bază referitoare la baze de date relaționale, limbajul SQL, funcționalități MySQL, precum și la posibilitatea de accesare a bazelor de date MySQL prin intermediul aplicațiilor scrise în limbajul Python (detalii...)
Data Science și Machine Learning - un proiect educațional dedicat celor începători în care se explică eșența noțiunilor de Data Science și Machine Learning precum și a elementelor de interacțiune și diferență dintre acestea (detalii...)
Features Engineering - un proiect educațional în care se exemplifică procedurile de analiză a caracteristicilor, de transformarea a caracteristicilor non-numerice, de detectarea a valorilor aberante și de procesare a datelor lipsă (detalii...)
Data Science - biblioteca Pandas - un proiect în care se exemplifică procedura de utilizarea a diferitor instrumente ale bibliotecii Pandas în procesul de analiza a datelor, precum si în procesul de prelucrare și manipulare a acestora spre a fi utilizate în elaborarea modelelor Machine Learning (detalii...)
Data Science - biblioteca NumPy - un proiect în care se exemplifică procedura de utilizarea a instrumentelor bibliotecii NumPy la conversia imaginilor în matrici numerice, precum și la realizarea diferitor calcule asupra datelor numerice spre a putea fi utilizate în elaborarea modelelor Machine Learning (detalii...)
Data Visualisation - biblioteca Matplotlib - un proiect de inițiere în procedura de utilizarea a instrumentelor bibliotecii Matplotlib la vizualizare grafică a diferitor date în scopul prelucrării acestora spre a putea fi utilizate în elaborarea modelelor Machine Learning (detalii...)
Data Visualisation - biblioteca Seaborn - un proiect de inițiere în procedura de utilizarea a instrumentelor bibliotecii Seaborn la vizualizare grafică statistică a diferitor date în scopul prelucrării acestora spre a putea fi utilizate în elaborarea modelelor Machine Learning (detalii...)
