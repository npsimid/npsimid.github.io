031 : Proiecte educaționale

Proiecte Python și Machine Learning utilizate cu studenții în procesul educațional

Python GUI: Tkinter - Music player - un proiect în care se elaborează o aplicație "Music player" și care are drept scop acumularea de abilități practice în dezvoltarea interfețelor grafice de utilizator (GUI) utilizând limbajul de programare Python și biblioteca Tkinter a acestuia
Data Science - biblioteca Pandas - un proiect în care se exemplifică procedura de utilizarea a diferitor instrumente ale bibliotecii Pandas în procesul de analiza a datelor, precum si în procesul de prelucrare și manipulare a acestora spre a fi utilizate în elaborarea modelelor Machine Learning
Data Science - biblioteca NumPy - un proiect în care se exemplifică procedura de utilizarea a instrumentelor bibliotecii NumPy la conversia imaginilor în matrici numerice, precum și la realizarea diferitor calcule asupra datelor numerice spre a putea fi utilizate în elaborarea modelelor Machine Learning
Machine Learning - Pregatirea datelor - un proiect în care se exemplifică procedura de pregatire a datelor pentru elaborarea unui model Machine Learning. Accentul se pune pe etapele de excludere a datelor lipsă și de transformare a datelor non-numerice în valori numerice. În procesul de pregatire a datelor se utilizeasă Pandas, NumPy și Scikit-Learn
Machine Learning - Probleme de clasificare - un proiect în care se exemplifică procedura de elaborarea a unui model Machine learning în cadrul problemelor de clasificare. Proiectul mai include etapele de evaluare a modelului și de realizarea a predicției utilizând instrumente Scikit-Learn
Machine Learning - Probleme de regresie - un proiect în care se exemplifică procedura de elaborarea a unui model Machine learning în cadrul problemelor de regresie. Proiectul mai include etapele de evaluare a modelului și de realizarea a predicției utilizând instrumente Scikit-Learn
Deep Learning - Bazele TensorFlow - un proiect educațional de familiarizare cu principalele noțiuni referitoare la tensori in mediul Google Colab
Va continua!!! Alte proiecte sunt in etapa de pregatire spre a putea fi încărcate în curând.
