Proiecte Machine learning

Proiecte Machine Learning utilizate cu studenții în procesul educațional

Proiecte Machine Learning generale

Machine Learning - Introducere - un proiect educațional dedicat celor începători prin intermediul căruia se realizeză o introducere în tehnologia Machine Learning explicându-se conceptele de bază și algoritmu de lucru în procesul de elaborare a modelelor (detalii...)
Machine Learning - Pregatirea datelor - un proiect în care se exemplifică procedura de pregatire a datelor pentru elaborarea unui model Machine Learning. Accentul se pune pe etapele de excludere a datelor lipsă și de transformare a datelor non-numerice în valori numerice. În procesul de pregatire a datelor se utilizeasă Pandas, NumPy și Scikit-Learn (detalii...)
Machine Learning - Sarcini de clasificare - un proiect în care se exemplifică procedura de elaborarea a unui model Machine learning în cadrul sarcinilor de clasificare. Proiectul mai include etapele de evaluare a modelului și de realizarea a predicției utilizând instrumente Scikit-Learn (detalii...)
Machine Learning - Sarcini de regresie - un proiect în care se exemplifică procedura de elaborarea a unui model Machine learning în cadrul sarcinilor de regresie. Proiectul mai include etapele de evaluare a modelului și de realizarea a predicției utilizând instrumente Scikit-Learn (detalii...)
Machine Learning - Cross Validation - un proiect în care se exemplifică procedura de utilizarea a mecanismelor de divizarea a datelor și de învătare a modelului prin diverse metode de reglarea a hiperparametrilor cu selectarea celui mai eficient model de predicție (detalii...)
